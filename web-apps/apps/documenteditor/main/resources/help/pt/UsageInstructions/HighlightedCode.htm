Inserir código destacado

No Editor de Documentos, você pode incorporar o código destacado com o estilo já ajustado de acordo com a linguagem de programação e estilo de coloração do programa que você escolheu.

  1. Vá para o seu documento e coloque o cursor no local onde deseja incluir o código.
  2. Mude para a guia Plug-ins e escolha
    Realçar código.
  3. Especifique o Idioma de programação.
  4. Selecione um Estilo do código para que apareça como se estivesse aberto neste programa.
  5. Especifique se deseja substituir tabulações por espaços.
  6. Escolha Cor de fundo. Para fazer isso, mova manualmente o cursor sobre a paleta ou insira o valor RGB/HSL/HEX.
  7. Clique em OK para inserir o código.
