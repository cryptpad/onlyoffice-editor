Inserir data e hora

Para inserir Data e hora no Editor de Documentos,

  1. coloque o cursor onde deseja inserir Data e hora,
  2. mude para a guia Inserir da barra de ferramentas superior,
  3. clique em Data e Hora
    ícone na barra de ferramentas superior,
  4. na janela Data e Hora que aparecerá, especifique os seguintes parâmetros:
    • Selecione o idioma necessário.
    • Selecione um dos formatos sugeridos.
    • Marque a caixa de seleção Atualizar automaticamente para permitir que a data e a hora sejam atualizadas automaticamente com base no estado atual.

      Observação: você também pode atualizar a data e a hora manualmente usando a opção Atualizar campo no menu contextual.

    • Clique no botão Definir como padrão para tornar o formato atual o padrão para este idioma.
  5. Clique no botão OK.
