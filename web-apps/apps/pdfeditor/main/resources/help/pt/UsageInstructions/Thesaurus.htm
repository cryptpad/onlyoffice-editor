Encontre um sinônimo

Se você estiver usando a mesma palavra várias vezes ou se uma palavra não for exatamente a palavra que você está procurando, ONLYOFFICE Editor de PDF permite que você procure sinônimos. Ele também mostrará os antônimos.

  1. Selecione a palavra no seu documento.
  2. Mude para a guia Plugins e escolha
    Dicionário de sinônimos.
  3. Os sinônimos e antônimos aparecerão na barra lateral esquerda.
