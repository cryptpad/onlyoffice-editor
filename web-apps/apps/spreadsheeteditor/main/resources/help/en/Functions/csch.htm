CSCH Function

The CSCH function is one of the math and trigonometry functions. It is used to return the hyperbolic cosecant of an angle specified in radians.

Syntax

CSCH(number)

The CSCH function has the following argument:

Argument Description
number The angle in radians that you wish to calculate the hyperbolic cosecant of. A numeric value entered manually or included into the cell you make reference to. Its absolute value must be less than 2^27.
