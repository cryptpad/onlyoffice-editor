DECIMAL Function

The DECIMAL function is one of the math and trigonometry functions. It is used to convert a text representation of a number in a given base into a decimal number.

Syntax

DECIMAL(text, radix)

The DECIMAL function has the following arguments:

Argument Description
text The text representation of the number you want to convert. The string lenght must be less than or equal to 255 characters.
radix The base of the number. An integer greater than or equal to 2 and less than or equal to 36.
