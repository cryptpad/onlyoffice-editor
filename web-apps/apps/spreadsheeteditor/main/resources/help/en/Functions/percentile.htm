PERCENTILE Function

The PERCENTILE function is one of the statistical functions. It is used to analyze the range of data and return the k-th percentile.

Syntax

PERCENTILE(array, k)

The PERCENTILE function has the following arguments:

Argument Description
array The selected range of cells for which you want to calculate the k-th percentile.
k The percentile value, a numeric value greater than or equal to 0 but less than or equal to 1.
