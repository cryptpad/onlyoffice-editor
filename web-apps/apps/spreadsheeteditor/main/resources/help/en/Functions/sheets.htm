SHEETS Function

The SHEETS function is one of the information functions. It is used to return the number of sheets in a reference.

Syntax

SHEETS([reference])

The SHEETS function has the following argument:

Argument Description
reference A reference for which you want to find out the number of sheets it contains. It is an optional argument. If reference is omitted, the number of sheets in the current workbook is returned.
