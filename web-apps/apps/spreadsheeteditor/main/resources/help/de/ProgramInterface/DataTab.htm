Registerkarte Daten

Mithilfe der Registerkarte Daten in der Tabellenkalkulation können Sie die Daten im Arbeitsblatt verwalten.

Die entsprechende Dialogbox in der Online-Tabellenkalkulation:

Die entsprechende Dialogbox in der Desktop-Tabellenkalkulation:

Sie können:

  • die Daten sortieren und filtern,
  • Text in Spalten konvertieren,
  • die Duplikate aus dem Datenbereich entfernen,
  • die Daten gruppieren und die Gruppierung aufheben,
  • Datenüberprüfungsparameter einstellen,
  • Daten aus einer Text-/CSV-Datei importieren,
  • Andere Dateien anzeigen, mit denen die Tabelle über die Schaltfläche Externe Links verknüpft ist.
