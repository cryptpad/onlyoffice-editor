Guia Dados

A guia Dados permite gerenciar dados no Editor de Planilhas.

A janela correspondente do Editor de Planilhas da área de trabalho:

Usando esta guia, você pode:

  • classificar e filtrar dados,
  • converter texto para colunas,
  • remover duplicatas de um intervalo de dados,
  • agrupar e desagrupar dados,
  • definir parâmetros de validação de dados,
  • adquirir dados do arquivo TXT/CSV.
