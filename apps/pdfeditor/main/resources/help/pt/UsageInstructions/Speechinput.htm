Digite por voz

No Editor do PDF , você pode inserir texto com sua voz.

  1. Coloque o cursor no local onde deseja adicionar texto.
  2. Mude para a guia Plugins e selecione
    Entrada de fala.
  3. Escolha o idioma de reconhecimento na janela pop-up.
  4. Clique no botão Iniciar ditado e comece a falar. Depois de fazer uma pausa, o texto será adicionado ao PDF. Para desligar o reconhecimento de voz, pressione o botão novamente.
    Para que o plugin funcione corretamente, você deve ter um dispositivo de entrada (por exemplo, um microfone ou fone de ouvido), bem como permissões para o navegador usar esses dispositivos de gravação.
