Editar HTML

Se você deseja obter texto como código HTML, use o plugin HTML.

  1. Abra a aba Plugins e clique em Obter e colar HTML.
  2. Selecione o conteúdo necessário.
  3. O código HTML do parágrafo selecionado será exibido no campo do plugin no painel esquerdo. Você pode editar o código para alterar as características do texto, por exemplo, tamanho da fonte ou família da fonte, etc.
  4. Clique no botão Copiar para copiar todo o código.
  5. Para limpar o campo do plugin, clique no botão Limpar.

Para obter mais informações sobre o plugin HTML e sua instalação, consulte o página do plugin no AppDirectory.
