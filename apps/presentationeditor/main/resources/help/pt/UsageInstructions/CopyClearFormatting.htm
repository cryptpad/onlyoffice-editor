Copiar/limpar formatação

Para copiar uma determinada formatação de texto no Editor de Apresentações,

  1. selecione a passagem de texto cuja formatação você precisa copiar com o mouse ou usando o teclado,
  2. clique no Copiar estilo
    ícone na aba Início da barra de ferramentas superior (o ponteiro do mouse ficará assim
    ),
  3. selecione a passagem de texto à qual deseja aplicar a mesma formatação.

Para aplicar a formatação copiada a várias passagens de texto,

  1. selecione a passagem de texto cuja formatação você precisa copiar com o mouse ou usando o teclado,
  2. clique duas vezes no ícone Copiar estilo
    na guia Início da barra de ferramentas superior (o ponteiro do mouse parecerá assim
    e o ícone Copiar estilo permanecerão selecionados:
    ),
  3. selecione as passagens de texto necessárias uma a uma para aplicar a mesma formatação a cada uma delas,
  4. para sair desse modo, clique no ícone Copiar estilo
    novamente ou pressione a tecla Esc no o teclado.

Para remover rapidamente a formatação que você aplicou a uma passagem de texto,

  1. selecione a passagem de texto cuja formatação deseja remover,
  2. clique no ícone Limpar estilo
    na guia Página Inicial da barra de ferramentas superior.
