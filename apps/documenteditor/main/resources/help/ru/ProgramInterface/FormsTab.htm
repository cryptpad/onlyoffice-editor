Вкладка Формы

эта вкладка доступна только для файлов DOCXF.

Вкладка Формы Редактора документов позволяет создавать в документах заполняемые формы, например, проекты договоров или опросы. Добавляйте и редактируйте текст и поля формы, чтобы создать заполняемую форму любой сложности.

Окно онлайн-редактора документов::

Окно десктопного редактора документов:

С помощью этой вкладки вы можете выполнить следующие действия:

  • Вставлять и редактировать
    • текстовые поля,
    • поля со списком,
    • выпадающие списки,
    • флажки,
    • переключатели,
    • изображения,
    • адреса e-mail,
    • номера телефона,
    • дату и время,
    • индексы,
    • номера кредитных карт,
    • составные поля,
  • очистить все поля и использовать настройки подсветки,
  • перемещаться по полям формы с помощью кнопок Предыдущее поле и Следующее поле,
  • просмотреть полученные формы в вашем документе,
  • управлять ролями,
  • сохранить форму как заполняемый файл OFORM.
