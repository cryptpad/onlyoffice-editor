Editar HTML

Se você está escrevendo uma página de site em um editor de texto e deseja obtê-la como um código HTML, use o plug-in HTML.

  1. Abra a guia Plug-ins e clique em Obter e colar html.
  2. Selecione o conteúdo necessário.
  3. O código HTML do parágrafo selecionado será exibido no campo do plug-in no painel esquerdo. Você pode editar o código para alterar as características do texto, por exemplo tamanho da fonte ou família da fonte etc.
  4. Clique em Colar no documento para inserir o texto com seu código HTML editado na posição atual do cursor em seu documento.

Você também pode escrever seu próprio código HTML (sem selecionar nenhum conteúdo do documento) e colá-lo em seu documento.

Para obter mais informações sobre o plug-in HTML e sua instalação, consulte o página do plugin no AppDirectory.
