ASIN Function

The ASIN function is one of the math and trigonometry functions. It is used to return the arcsine of a number.

Syntax

ASIN(number)

The ASIN function has the following argument:

Argument Description
number The sine of the angle you wish to find, a numeric value greater than or equal to -1 but less than or equal to 1 entered manually or included into the cell you make reference to.
