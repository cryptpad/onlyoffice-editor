Guia Layout

O que é a guia Layout?

A guia Layout no Editor de Planilhas permite ajustar a aparência de uma planilha: configurar os parâmetros da página e definir a disposição dos elementos visuais.

Funções da guia Layout

Usando esta guia, você pode:

  • ajustar página margens, orientação, tamanho,
  • especifique um área de impressão,
  • inserir cabeçalhos ou rodapés,
  • dimensionar uma planilha,
  • imprimir títulos em uma página,
  • alinhar e organizar objetos (imagens, charts, formas),
  • mudança esquema de cores.
